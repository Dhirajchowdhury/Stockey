### Technical Indicators Calculation

#### RSI (Relative Strength Index)
```
1. Calculate price changes over 14 periods
2. Separate gains and losses
3. Calculate average gain and average loss
4. RS = Average Gain / Average Loss
5. RSI = 100 - (100 / (1 + RS))

Interpretation:
- RSI > 70: Overbought (potential sell)
- RSI < 30: Oversold (potential buy)
- RSI 30-70: Neutral
```

#### MACD (Moving Average Convergence Divergence)
```
1. Calculate 12-period EMA
2. Calculate 26-period EMA
3. MACD Line = EMA(12) - EMA(26)
4. Signal Line = 9-period EMA of MACD Line
5. Histogram = MACD Line - Signal Line

Interpretation:
- Histogram > 0: Bullish momentum
- Histogram < 0: Bearish momentum
- MACD crosses above Signal: Buy signal
- MACD crosses below Signal: Sell signal
```

#### Moving Averages
```
Simple MA = Sum of prices / Number of periods

Exponential MA:
1. Calculate SMA for initial period
2. Calculate multiplier = 2 / (period + 1)
3. EMA = (Current Price - Previous EMA) Ã— Multiplier + Previous EMA

Interpretation:
- Price > MA: Uptrend
- Price < MA: Downtrend
- MA7 > MA30: Golden Cross (bullish)
- MA7 < MA30: Death Cross (bearish)
```

### Database Optimization

```
Indexes:
- User: email, googleId
- Stock: symbol, name (text), changePercent, volume
- StockPriceHistory: (symbol, date), (symbol, interval, date)
- News: publishedAt, relatedStocks, sentiment.label
- Prediction: (symbol, metadata.generatedAt)

Query Optimization:
- Use .lean() for read-only queries
- Select only needed fields
- Limit results with pagination
- Use compound indexes for complex queries
```
